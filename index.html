    ❓ Où trouver mon fichier .sav ?

    Le jeu sauvegarde ta progression dans un fichier CryptCustodian1.sav (ou 2, 3 selon ton slot de save). Voici où il se trouve selon ton OS :

    ⚠️ Important avant d'importer

    • L'import est lecture seule, ton fichier n'est pas modifié
    • Le fichier à importer est CryptCustodian1.sav (ou 2/3) — pas Settings.sav ni paint1.sav

    🪟 Windows

    Ouvre l'Explorateur de fichiers, copie-colle dans la barre d'adresse :

    %LOCALAPPDATA%\CryptCustodian\CRYPTCUSTODIAN

    Ou chemin complet : C:\Users\TonNom\AppData\Local\CryptCustodian\CRYPTCUSTODIAN\

    🍎 macOS

    Ouvre le Finder, puis menu Aller → Aller au dossier… (Cmd+Maj+G) et tape :

    ~/Library/Application Support/com.gamemaker.cryptcustodian

    Le nom du dossier peut varier légèrement (cherche "Crypt" dans ~/Library/Application Support/)

    🐧 Linux

    Ouvre un terminal et navigue vers :

    ~/.config/CryptCustodian/

    Sinon essaye ~/.local/share/CryptCustodian/

    🎮 Steam Deck / Proton

    Le jeu tourne sous Proton, donc la save est dans le préfixe Wine :

    ~/.steam/steam/steamapps/compatdata/2394650/pfx/drive_c/users/steamuser/AppData/Local/CryptCustodian/CRYPTCUSTODIAN/

    2394650 = Steam App ID de Crypt Custodian
